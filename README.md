# Chronos

A Pomodoro timer and task management app built with React 19 and TypeScript.

## Stack

- **React 19** + **TypeScript** + **Vite 8**
- **Tailwind CSS v4** + **shadcn/ui**
- **React Router v7**, **TanStack Query v5**, **framer-motion**

## Development

```bash
bun dev          # Start dev server
bun run build    # Type-check + production build
bun run lint     # Lint
bun run preview  # Preview production build
```
